# Gitignore

Create .gitignore files from <https://www.gitignore.io> templates.

## Install

With [fisherman]

```
fisher sijad/gitignore
```

## Usage

```fish
gitignore osx c++
```

## Options

* -u, --update:
    Update template list.

* -o, --output=*file*:
    Save created template to *file*.

* -h, --help:
    Show usage help.
