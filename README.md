# Gitignore

Create `.gitignore` files from <https://www.gitignore.io> templates.

## Install

With [Fisher](https://github.com/jorgebucaran/fisher)

```
fisher add sijad/gitignore
```

## Usage

```fish
gitignore osx c++
```

## Options

* `-u`, `--update`:
    Update template list.

* `-o`, `--output=FILE`:
    Save generated template to `FILE`.

* `-h`, `--help`:
    Show usage help.
